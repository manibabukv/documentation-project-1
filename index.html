Introduction:

As a part of this project, I need to re-create John Snow’s map of London’s 1854 Cholera Epidemic Outbreak using D3. D3 is a JavaScript library for creating solid and interactive visualizations. Also, need to create additional supporting charts for the given data and enable interaction between map and charts.

Design Choices:

I wanted my background to be white, so that everything that I added on my page will be clearly visible.

 

Titles and other Text:

For titles and other text on the visualization I used black color because it will be clearly visible on the white background.

 

Map:

For street lines I used black color which is the most visible color on white background. Keeping in mind the color blindness I tried to make color choices based on. The same reason goes for street names and other labels. I applied black stroke to pumps, and landmarks to highlight them. The same goes with victims.

 

Zoom and Filters:

For zoom buttons I used default html button and for filters I used radio buttons with green accent color. Buttons enable us to interact easily with our visualization and it gives clear idea to the viewer about its purpose.

 

Stacked Bar Chart:

The main reason for choosing stacked bar chart as my choice of chart is it is suitable for grouping data based on various categories and it helps us in filtering victims in all possible ways we can filter when we combine it with radio button filters. The color palette I used for the stacked bar chart is based on various visualization factors.

 

Bar Chart:

I chose bar chart for representing deaths over dates. The reason behind this is bar chart is best for numerical data. It is easy to understand when you compare it with other charts because of better data visibility. I fill the rectangles of bar chart with cadet blue color.

 

Tooltip:

For tooltip I used sky blue background with black color text which will be clearly visible on this background. I also added a black boarder so that it does overlap with other visuals.
